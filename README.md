# Учебники: Разработка функций в Python (Feature Engineering)
[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/rasgointelligence/feature-engineering-tutorials/main) [![Chat on Discourse](https://img.shields.io/discourse/status?server=https%3A%2F%2Fforum.rasgoml.com)](https://forum.rasgoml.com/) [![Chat on Slack](https://img.shields.io/badge/chat-on%20Slack-brightgreen.svg)](https://join.slack.com/t/rasgousergroup/shared_invite/zt-nytkq6np-ANEJvbUSbT2Gkvc8JICp3g)

Эндрю Нг говорил, что «прикладное машинное обучение - это, по сути, просто разработка функций». В науке о данных и машинном обучении наиболее важной, но часто игнорируемой частью головоломки является разработка функций.

В [Rasgo](https://www.rasgoml.com/) специалисты по анализу данных разраюотали ряд инструментов, цель которых - дать возможность глобальному сообществу специалистов по науке о данных генерировать ценные и достоверные выводы из данных менее чем за 5 минут. Работая в этой области можно констатировать малое кол-во полезного контента и функций Python, предназначенных для разработки функций. Разработчики сталкиваемся с этими проблемами каждый день, и мы хотели предоставить репозиторий рецептов, демонстрирующих, как использовать лучшие инструменты, доступные в этой сфере. SDK [(Rasgo)](https://www.rasgoml.com/) позволяет пользователям автоматически отслеживать, визуализировать и оценивать свои эксперименты по разработке функций, чтобы принимать более точные и объяснимые решения по разработке функций.

Этот репозиторий содержит учебные пособия и код, позволяющие специалистам по обработке данных легко создавать новые функции машинного обучения и оценивать их важность для контролируемого машинного обучения.
